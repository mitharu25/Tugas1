1. < h >
Tag Heading berfungsi untuk membuat heading pada artikel dengan berbagai ukuran (h1,h2,h3,h4,h5,h6).

2. < b >
Tag bold berfungsi untuk huruf bercetak tebal.

3. < p > & < br >
Tag < p > untuk membuat paragraf dan Tag < br > untuk membuat garis baru.

4. < i >
Tag i berfungsi untuk Membuat teks miring.

5. < head > & < body >
Tag head berfungsi untuk Informasi meta tentang dokumen.
Tag body berfungsi untuk tempat dibuatnya semua konten website menggunakan HTML

6. < ul > & < li >
Tag ul berfungsi untuk membuat daftar dengan selain nomor.
Tag li berfungsi untuk membuat sebuah item daftar.

  • Apel
  • Jeruk
  • Melon

7. < ol >
Tag ol berfungsi untuk membuat daftar dengan nomor.

  1. SD (Sekolah Dasar)
  2. SMP (Sekolah Menengah Pertama)
  3. SMA (Sekolah Menengah Atas)

8. < small >
Tag small berfungsi untuk membuat teks kecil.

9. < hr >

Tag hr berfungsi untuk Memisahkan konten (biasanya ditampilkan garis pembatas).

10. < sub >
Tag sub berfungsi untuk membuat Teks subscript.

11. < sup >
Tag sub berfungsi untuk membuat Teks superscript.

12. < button >
Tag button berfungsi untuk membuat button/tombol pada web.

13. < input >
Tag input berfungsi untuk membuat tipe inputan pada form yang akan dibuat.

14. < textarea >
Tag textarea berfungsi membuat Elemen untuk mendefinisikan field input.

15. < fieldset >
Tag fieldset berfungsi untuk Mengelompokan elemen yang terdapat pada sebuah form.

semangka
jeruk
satu
dua

16. < select >
Tag select berfungsi untuk Membuat input dengan pilihan yang berbentuk list drop down.

19. < img > & src
Tag image berfungsi membuat Elemen untuk mendefinisikan gambar, sedangkan src untuk Atribut untuk menentukan URL gambar.

20. < audio >
Tag audio berfungsi untuk membuat audio player dan memutarkan audio tersebut.

21. < address >
Tag address berfungsi untuk mendifinisikan alamat mulai dari alamat fisik, alamat email, nomor telepon, URL, sosial media, dan lain sebagainya.
